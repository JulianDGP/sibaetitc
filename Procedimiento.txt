Pre-requisitos:
Se debe tener instalado python 3.7, pip3, Virtualenv
1. Creación del proyecto PyCharm, configuración del github y creación del proyecto Django
1.1 Se crea un nuevo proyecto en PyCharm llamado Proyecto_SIBAETITC y se expande la pestaña Project interpreter: new virtual enviorement y en new environmentusing se selecciona Virtualenv.
1.2 En github desktop creamos un nuevo repositorio llamado sibaetitc, descripción: Sistema de información para gestionar los servicios de alimentación del banco de alimentos de la Escuela Tecnológica Instituto Técnico Central, dentro del directorio Proyecto_SIBAETTIC seleccionando gitignore python. Se abre el .gitignore y se comenta la linea que dice db.sqlite3,luego se crea un archivo de texto llamado requirements.txt que contendrá:
Django==2.2.7
psycopg2==2.8.4
pytz==2019.3
sqlparse==0.3.0
